## Das OpenSIN Brand System — bevor ein Pixel generiert wird

### Farben die jeder erkennt
| Farbe | Hex | Wo sie verwendet wird |
|-------|-----|----------------------|
| **OpenSIN Grün** | `#00bb7f` | CTAs, Highlights, Subtitle Highlights |
| **Dark Background** | `#09090b` | Alle Images und Videos |
| **Accent Red** | `#ff2357` | Alerts, Emphasis, Secondary CTAs |
| **Tech Blue** | `#54a2ff` | Data Streams, Connections |

### Typography die konsistent ist
- **Primary:** Inter — überall. Blog, Video, Social, Docs.
- **Monospace:** JetBrains Mono — für Code Examples.
- **Video Subtitles:** Inter 600 weight, 1.5rem, mit Green Highlights für Key Terms.

### Brand Voice die nicht nach AI klingt
- **Tone:** Direct. Premium. Confident. Technical aber zugänglich.
- **Message Hierarchy:** Hook → Promise → Proof → CTA. Immer.
- **Banned Words:** "magic", "revolutionary", "game-changing", "disrupt", "cutting-edge", "next-gen"
- **Preferred Words:** "enterprise", "orchestrate", "autonomous", "production-grade", "scalable"

## Die 8-Step Pipeline die alles verändert

<p>Image Placeholder</p>

*Von Brief bis Publication — 8 Steps. Jeder enforced Brand Compliance.*

### Step 1: Brief
Nicht "mach ein Bild". Sondern:
- **Audience:** Enterprise CTOs, 35-50, technisch versiert
- **Key Message:** 9 spezialisierte Agenten für production-grade AI Workflows
- **Channel:** Blog (langform, technisch, mit Code Examples)
- **Format:** Article mit eingebetteten Images und Videos

### Step 2: Script
Das Script folgt der Message Hierarchy:
```
HOOK: "89% der Menschen vertrauen einer Brand nicht wenn die Videos schlecht aussehen..."
PROMISE: "OpenSIN löst das mit einer kompletten Brand Media Pipeline..."
PROOF: "Gemini Nano Banana Pro, Veo 3, TTS, Brand Enforcer mit 10+ Rules..."
CTA: "Besuche opensin.ai und sieh es selbst."
```

### Step 3: Generate Raw Media
**Für Images:** Gemini Nano Banana Pro (Imagen 4) mit brand-compliant Prompt:
```
A stunning blog header for OpenSIN AI. Dark background (#09090b) with 
glowing green (#00bb7f) neural network connections. Clean, modern, 
enterprise-grade design. No text in image. 16:9 aspect ratio.
```

**Für Videos:** Veo 3.0 Generate mit cinematic Prompt:
```
A cinematic shot of a futuristic AI dashboard with glowing green data 
streams. Neural network nodes pulse with light. Professional tech 
aesthetic, photorealistic, 16:9.
```

### Step 4: TTS Narration
**Gemini 2.5 Flash Preview TTS** mit Aoede Voice:
- 24kHz PCM Quality
- Natural Pausen und Emphasis
- Getimed zur Video Duration

### Step 5: Subtitles
SRT File generiert aus dem Script:
- Modern minimal Style
- Grün (#00bb7f) Highlights für Key Terms
- Max 42 Characters pro Line, 2 Lines max
- Word-by-word Animation

### Step 6: Packaging
Hier wird aus generischem Content **OpenSIN Content**:
- ✅ Logo Watermark auf jedem Frame (top-right, 5%, 80% opacity)
- ✅ Subtitles gerendert mit Brand Styling
- ✅ End Card appended (3 Sekunden: Logo + "Visit opensin.ai")
- ✅ Color Grading auf Brand Palette
- ✅ BGM: Ambient electronic bei 15% Volume

### Step 7: Brand Review
Der **Brand Enforcer** checkt 10+ Rules:
```typescript
interface BrandCheckResult {
  passed: boolean
  violations: BrandViolation[]
  score: number // 0-100 — kein Asset published unter 85
}
```

### Step 8: Publish
Platform-spezifische Exports:
- **YouTube:** 16:9, H.264, mit Subtitles
- **TikTok:** 9:16, H.264, mit Subtitles
- **Instagram:** 1:1, JPEG, mit Logo
- **LinkedIn:** 16:9, JPEG, mit Logo
- **Blog:** Markdown mit eingebetteten GitLab URLs
