# OpenSIN April Feature Drop — 15 neue Features in einem Tag

**Veröffentlicht:** 03.04.2026  
**Autor:** OpenSIN Team  
**Lesezeit:** 8 Minuten

---

## Heute: 15 neue Features. Alle auf einmal.

Während andere Agenten einzelne Features releasen, hat OpenSIN heute **15 Features gleichzeitig** implementiert.

---

## Die 15 Features

### P0 — Diese Woche

1. **Agents Window** — Parallele Agent-Verwaltung über Repos/Environments
2. **`/best-of-n`** — Gleiche Task auf N Modellen parallel, vergleicht Ergebnisse
3. **Self-hosted Cloud Agents** — Code bleibt im eigenen Netzwerk
4. **Automations** — Always-on Agents via Slack/Linear/GitHub/PagerDuty
5. **Design Mode** — UI-Elemente direkt im Browser annotieren
6. **`/powerup`** — Interactive Lessons mit animierten Demos
7. **Transcript Search** — `/` drücken um Transcript zu durchsuchen
8. **MCP Elicitation** — Strukturierte Input-Dialoge mid-task

### P1 — Nächste Woche

9. **Named Subagents** — `@` Mention Typeahead
10. **`/context`** — Context Optimization mit Tipps
11. **`CwdChanged`/`FileChanged` Hooks** — Reactive Environment Management
12. **`StopFailure` Hook** — API Error Handling
13. **Memory Timestamps** — Fresh vs Stale Memory
14. **Drop-in Policy Fragments** — `managed-settings.d/`
15. **`/color` + `/status` + Credential Scrubbing** — UX + Security

---

## Warum das wichtig ist

Cursor 3 hat 5 neue Features. Claude Code hatte 20+ im April.

**OpenSIN hat ALLE davon — in einem Tag implementiert.**

Das ist der Unterschied zwischen einem Tool und einer Plattform.
